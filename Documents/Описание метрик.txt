Vusers:

Running Vusers - Количество запущенных пользователей и их состояние на протяжении всего теста.
Vusers Summary - Общее количество пользователей и статус их отработки.

Errors:

Error Statistics (by Description) - Показывает количество ошибок по ходу запуска теста и группирует их по типу ошибки.
Error per Second (by Description) - Показывает среднее количество ошибок ежесекундно по ходу теста и группирует их по типу ошибки.
Error Statistics - Показывает количество ошибок по ходу запуска теста и группирует их по коду ошибки.
Errors per Second - Показывает среднее количество ошибок ежесекундно и группирует их по коду ошибки.
Total Errors per Second - Показывает общее количество ошибок за каждую секунду теста.
Total Errors - Общее количество ошибок в отработанном тесте.

Transactions:

Average Transactions Response Time - Среднее время ответов сервера.
Total Transactions per Second - Общее количество транзакций/сек. за тест.
Total Passed Transactions - Общее количество проведённых транзакций за тест.
Transaction Summary - Результат успешности всех проведённых транзакций и их общее количество
Transaction Performance Summary - Статистика временён выполнения всех транзакций сценария мин.\авг.\макс.

Web Resources:

Hits per Second - График демонстрирует количество запросов пользователей на сервер/сек.
Throughput - Статистика пропускной способности тестируемой системы. В байтах.
Throughput (MB) - -//- . В Мб.
HTTP Status Code Summary - Статистика всех ответов сервера за тест и их статус.
HTTP Responses per Second - График ответов сервера/сек. за время теста.
Retries Summary - Количество ретраев соединения с сервером за время сценария и их группирование по причинам.
Connections - График созданных соединений с сервером за всё время исполнения сценария.
Connections Per Second - График создания и закрытия подключений к верверу/сек. за всё время исполнения сценария.


Windows - основные счётчики процессора:

% Processor Time - Процент времени которое ЦП находится в активном состоянии.
% User Time - Процент времени затреченное ЦП на пользовательские задачи.
Processor Queue Length - Длина очереди потоков, ждущих пока процессор освободится.

Windows - основные счётчики памяти:

Available MBytes - Количество доступной физической памяти для работы процессов.
% Committed Bytes In Use - Соотношение выделенной памяти с лимитом для выделения памяти.
Pages Faults/sec - Количество отказов страниц/сек. 
Pages/sec - График интенсивности обмена с файлом подкачки.

Windows - основные счётчики ввода/вывода:

Avg. Disk secs/Read - Среднее время чтения данных с диска.
Avg. Disk secs/Write -  Среднее время записи данных на диск.
Avg. Disk Queue Length - Cредняя длина очереди запросов к диску.
Current Disk Queue Length - Текущая длина очереди запросов к диску
% Disk Time - Показывает процент общей загруженности диска.

Windows - основные сетевые счетчики:

Bytes Total/sec - Показывает объем входящего и исходящего трафика.
BytesReceived/sec (в Grafana) - Получено байт/с.
BytesSent/sec (в Grafana) - Отправлено байт/с.
